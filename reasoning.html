Basic Set Theory

  • Ques: Define a set, and explain subsets and universal sets.


  • Set:
    A collection of distinct objects, often represented as A={1,2,3}.

  • Subset:
    A set B is a subset of A if every element of B is in A (denoted B⊆A).

  • Universal Set (U):
    Contains all elements under consideration, with subsets derived from it. Example: If U={1,2,3,4}, then A={1,2}⊆U.

Logical Statements

    A logical statement is a sentence that can be either true or false.

  • Truth Value: Indicates whether the statement is true (T) or false (F).

    • Examples:
  • "The sky is blue" (T)
  • "2 + 2 = 5" (F)

Types of Logical Connectives

  • NOT (¬): Reverses the truth value of a statement.
  • AND (∧): True if both statements are true.
  • OR (∨): True if at least one statement is true.
  • IMPLIES (→): True unless a true statement implies a false one.
  • IF AND ONLY IF (↔): True if both statements are either true or false.

Logical Equivalence

  • Two statements are logically equivalent if they always have the same truth values.
  • Example: p→q is logically equivalent to its contrapositive ¬q→¬p.
